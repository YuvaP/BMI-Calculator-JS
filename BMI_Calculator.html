Yuva BMI Calculator for Body Mass Index Calculation using the below Formula.
BMI = (weight/height*height)*703



Goal: Calculate your Body Mass Index value.

How does this BMI app work?

  • The user inputs the Height and Weight.
  • BMI value is calculated using the BMI formula shown above.
  • Based on the standard health data, the user gets the result with suggestions.
  • The user can clear the result and try again.
